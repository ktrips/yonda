設定のヘルプ

Yondaの使い方は別ページでご確認ください。

アプリ設定

メニュー → アプリ設定 で以下を設定できます。

  • デフォルトのページ：起動時に表示するページ（Yonda / Yomu / Oshi）
  • アフィリエイトタグ：Kindle・Audible リンクに付与するタグ
  • AI プロバイダ・API キー：写真から本検索、AI推しで使用

世田谷区立図書館

  • メニューから「アカウント設定」を開く
  • 利用者番号とパスワードを入力
  • 「ログインテスト & 保存」で接続確認

お気に入り登録済みの本は、貸出履歴取得時に自動で「♥ お気に入り」として表示されます。

Audible Japan

  • メニューから「アカウント設定」を開き、ソースを「Audible Japan」に選択
  • audible-cli で認証し、auth_jp.json をアップロードするか、yonda/data/auth_jp.json に配置
  • 「接続確認」で認証が成功すれば設定完了
  • 認証が期限切れの場合は audible-cli で再認証し、ファイルを再アップロード

Kindle

方法1: Amazon にログイン（推奨）

  • メニュー → アカウント設定でソース「Kindle」を選択
  • Amazon メールアドレスとパスワードを入力し「ログインテスト & 保存」
  • 2段階認証（OTP）が有効な場合、「読書記録を取得」時に OTP 入力画面が表示されます
  • OTP を入力して「OTP を送信して取得」をクリック
  • セッション永続化：OTP 認証後のセッションが自動保存され、次回以降は OTP なしで取得できます（有効期間: 約7日）
  • セッションが切れた場合のみ、再度 OTP 入力が必要になります

方法2: ローカルファイル

  • Kindle for Mac/PC をインストールし、蔵書を同期しておく
  • データファイルが自動検出されれば、認証なしで取得可能
  • 検出されない場合は、環境変数 YONDA_KINDLE_SQLITE_PATH または YONDA_KINDLE_XML_PATH でパスを指定

方法3: 認証 + ローカルフォールバック

  • 認証情報を登録したまま、Kindle for Mac を起動して蔵書を同期しておく
  • Amazon API が失敗した場合、自動でローカルファイル（BookData.sqlite）から取得されます

定期自動取得（Cloud Run 環境）

Cloud Run にデプロイした場合、Cloud Scheduler により毎日自動でデータが取得・保存されます。

  • 朝 06:00 / 昼 12:00 / 夕 18:00（JST）に図書館・Audible・Kindle を一括取得
  • 取得結果は GCS バケットに自動保存されます
  • 一部ソースが失敗しても他ソースの取得は継続されます
  • メニューの「読書記録取込み」で、ソースごとの最終取得日時を確認できます

Kindle のクラウド自動取得

  • Amazon メール・パスワードを登録し、初回のみ手動で OTP 入力して取得を実行します
  • 認証後セッションが GCS に自動保存され、以降は Cloud Scheduler が OTP なしで自動取得します（有効期間: 約7日）
  • セッション期限切れ後は、再度手動で OTP 入力が必要です
  • または BookData.sqlite を GCS バケットに配置し、YONDA_KINDLE_SQLITE_PATH=/mnt/data/BookData.sqlite を設定する方法でも取得可能

AI推し・写真検索（APIエラーが出る場合）

AI推し（５質問で推し、MBTI診断で推し、強み診断で推し）や写真から本を検索するには、メニュー → アプリ設定 で API キーを登録してください。

  • 401 エラー: APIキーが無効。Google AI Studio または OpenAI で新しいキーを発行し、再設定
  • 403 エラー: APIが有効化されていない、または課金設定が必要
  • 429 エラー: リクエスト制限超過。数十秒〜数分待ってから再試行
  • 400 エラー: 無料枠の地域制限の可能性。課金を有効化するか、別プロバイダ（OpenAI ↔ Gemini）に切り替えて試す

詳細は docs/AI_TROUBLESHOOTING.md を参照してください。

AI の利用状況確認：OpenAI Usage、Google Cloud Billing

認証情報はこのPC上のローカルファイルに保存され、外部には送信されません。
