## 未来のプレスリリース（2026 年 12 月想定）

> ### 手入力ゼロの読書記録 Yonda、読了自動記録の継続率を可視化
> ### ― 記録が 3 日で終わっていた読書家が、1 年分の読了リストを持てるようになった ―
>
> **2026 年 12 月 1 日** — Yonda は、Audible・Kindle・公共図書館・紙の本の読書記録を毎朝自動で同期する読書記録サービス。ユーザーが入力する項目は原則ゼロ。読了した本は AI が書評ポイントを生成し、次に読む本まで提案する。
>
> 読書記録が続かない要因は「読書量」ではなく「記録の手間」にある。Yonda は 3 つの仕組みでこれを取り除いた。
>
> 1. **記録ゼロ手入力**: Audible・Kindle・図書館の記録を毎朝自動同期。紙の本はカメラで撮るだけで AI がタイトルを抽出する。ユーザーが入力する項目は原則ゼロ。
> 2. **併用前提の設計**: 「電子書籍アプリ 1 つ」ではなく、Audible で聴き・Kindle で読み・図書館で借りる — サービスを横断する読書家のために作られている。
> 3. **読書が、あなたの財産に**: 読了リスト・年間集計・ジャンル傾向が自動で貯まり、読書がひとつの本棚として積み上がっていく。
>
> **ユーザーの声**: 「Audible と Kindle と図書館を使い分けていて、読んでいるのに何を読んだか思い出せなかった。Yonda は登録したら勝手に貯まっていくので、辞めるきっかけ自体がなかった」（40 代・会社員）
>
> Yonda は無料で今日から始められる。yonda.ktrips.net で「Google でログイン」から。

## 購入前に顧客が聞く FAQ 10 個

### Q1. 無料で何ができますか？

読書記録の自動同期・読了リスト・年間/ジャンル別の統計・AI 書評（一定数まで）・みんなの読書記録の閲覧が **すべて無料** です。クレジットカードの登録も不要です。

→ **仕様**: 習慣化の核（自動同期・読了リスト・統計）は無料枠で完結させる。有料化しても核は無料のまま。

### Q2. どのサービスに対応していますか？

Audible（日本）・Kindle・公共図書館（世田谷区立に対応、順次拡大）・紙の本（カメラ＋AI でタイトル抽出）に対応しています。1 つだけの利用でも、複数併用でも使えます。

→ **仕様**: 対応サービスは「増やす」より「併用体験」を訴求。図書館は汎用アダプタ設計なので追加は設定作業。

### Q3. 自分の地域の図書館は使えますか？

現在は世田谷区立図書館に対応しています。他の図書館は汎用アダプタ設計により順次追加予定です。使いたい図書館があればリクエストできます。

→ **仕様**: 未対応でも「リクエストできる」導線を用意（ニッチ拡張の需要計測にもなる）。

### Q4. 認証情報（図書館のパスワードなど）は安全ですか？

図書館・Amazon の認証情報は暗号化して保存し、同期の実行にのみ使用します。第三者への提供・販売は一切行いません。Audible は公式の認証ファイル方式を使い、パスワードそのものは保存しません。

→ **仕様**: 認証情報は暗号化保存・目的外利用なし・第三者提供なしを明記。プライバシーポリシーの中核。

### Q5. 読書データはどこに保存されますか？他人に見られませんか？

あなたの読書記録はあなたのアカウントにのみ保存されます。「みんなの読書記録」に公開されるのは、あなたが選んだ場合の読了数・本のタイトル程度で、認証情報や個人を特定する情報は共有されません。

→ **仕様**: 公開範囲は「読了数・タイトル」まで。デフォルトの公開範囲を明示し、オプトアウト可能に。

### Q6. データはエクスポートできますか？

読書記録は JSON 形式でダウンロードできます（ソース別）。あなたのデータはあなたのものです。ロックインしません。

→ **仕様**: エクスポートを保証（既に `/api/download/<source>` で実装済み）。「ロックインしない」は無名サービスへの信頼の担保。

### Q7. 月額料金の元は取れますか？（有料プラン導入時）

有料プラン（Plus）は月額 300〜500 円を想定しています。中心価値は AI 書評の無制限生成・同期頻度アップ・読書傾向の深掘り分析です。7 日間の無料トライアルで判断できます。書籍 1 冊分以下の金額です。

→ **仕様**: 価格は月 300〜500 円。Plus 特典は「原価がかかるもの（AI 呼び出し）・ヘビーユーザーだけが欲しがるもの」に限定。無料トライアル 7 日。

### Q8. 解約は簡単ですか？

いつでも 1 タップで解約でき、当社への連絡は不要です。解約後も無料機能と過去の記録はそのまま使えます。

→ **仕様**: 解約容易性を「先に」言う。サブスク疲れした顧客への最大の反証。解約後もデータと無料機能は残す。

### Q9. AI の書評や選書には、自分で API キーを用意する必要がありますか？

いいえ。**Plus では、AI 機能はこちらで用意します。** あなたが API キーを取得したり設定したりする必要はありません。（※現在は移行途中で一部設定が必要な場合がありますが、サーバー側代理呼び出しに統一予定です。）

→ **仕様（P2 最優先の直し）**: 「ユーザーが API キーを入れる」方式は作り手都合。原価を Plus 料金に織り込み、サーバー側で代理呼び出しする。

### Q10. 無名のサービスですが、すぐ終わりませんか？

Yonda は個人開発ですが、固定費が月数百円のため、少数のユーザーでも継続できる構造になっています。開発の記録は書籍・Web で公開しており、運営の顔が見えます。読了データはいつでもエクスポートできます。

→ **仕様**: 継続可能なコスト構造・運営の透明性・エクスポート保証の 3 点で「サ終不安」に反証。

## この FAQ から確定した設計変数（まとめ）

| 変数 | 確定値 |
|---|---|
| 無料枠 | 自動同期・読了リスト・統計・AI 書評（一定数）・コミュニティ閲覧・エクスポート |
| Plus 価格 | 月 300〜500 円、7 日無料トライアル |
| Plus 特典 | AI 書評無制限・同期頻度アップ・傾向深掘り分析（＝原価がかかる/ヘビーユーザー向け） |
| AI キー | サーバー側代理呼び出しに統一（ユーザーに API キーを要求しない） |
| プライバシー | 認証情報は暗号化・目的外利用なし・第三者提供なし。公開は読了数/タイトルまで、オプトアウト可 |
| 解約 | 1 タップ・連絡不要・解約後もデータと無料機能は残る |
| データ所有 | JSON エクスポートを保証、ロックインしない |
